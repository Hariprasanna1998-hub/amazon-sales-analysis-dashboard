# Amazon Sales Dashboard - Power BI

## Project Overview
This project analyzes Amazon sales data using Power BI to understand sales performance, profitability, and regional trends. The dashboard provides interactive visualizations to identify key business insights and support decision-making.

## Tools Used
- Power BI
- Data Visualization
- Business Intelligence

## Dashboard Features
- Total Sales, Profit, Quantity, and Shipment Cost KPIs
- Sales analysis by Category and Sub-Category
- Regional sales and profit comparison
- Yearly sales trend analysis
- Geographic visualization of profit by state

## Key Insights
- Standard class is the most preferred shipping mode among customers.
- West region generates the highest profit.
- Technology is the highest selling product category.
- In 2017, sales reached approximately 0.7M, the highest in the dataset.

## Recommendations
- Increase inventory for high-performing sub-categories:
  - Phones in Technology
  - Chairs in Furniture
  - Storage in Office Supplies

- Improve sales in South and Central regions by offering more discounts and promotional offers.
